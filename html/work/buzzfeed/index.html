Buzzfeed

 

Buzzfeed asked me to design and implement a new version of their mobile site optimized for the iPhone and Android platforms. We significantly pared down the feature set of the main site to create a focused experience for users on the go.

Visit Site ›

I crafted a clean interface, informed by Buzzfeed’s red and blue coloring and the emerging conventions around mobile sites. Everything was kept small so that load times would be fast even over slow cellular connections.

One of my minor accomplishments was setting images up so that they would grow in size when the device is rotated to landscape.
